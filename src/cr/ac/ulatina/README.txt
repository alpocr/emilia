Emilia Framwork es el resultado de un proyecto universitario
del curso Programación con Patrones.

El Framework básicamente ayuda a otros desarrolladores a crear aplicaciones
encargados de analizar el comportamiento estadístico de un grupo de valores.

El Framework posee una librería de cálculo básicos que puede usar para
sus proyectos; usted podrá generar sus propios cálculos y sus propios
tipos de gráficos. En la siguiente numeración se mencionan algunas
características.


Características:
1- Genera sieres lineales y de series de tiempo.
2- Librería básica con cálculos básicos de pronósticos.
3- Precisión de los datos.
4- Importar mediante CSV, XML.
5- Personalizar los gráficos.
6- Posibilidad de crear sus propios cálculos estadísticos y graficarlos. ##
